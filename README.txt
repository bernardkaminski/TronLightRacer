4.  Software Dependencies

Python 2.7 or higher (32 Bit) and PyGTK library.

5.	Software Platforms

This software was tested on Linux (Ubuntu), Mac OS and Windows using PyCharm 3 IDE. No compiler version, since Python is an interpreted language. Python 2.7 interpreter.

6.	Third-party libraries

PyGTK
URL: http://www.pygtk.org/ 
Version: 2.24.2
Installation instructions: 

LINUX
PyGTK is included in most Linux distributions (including Debian, Fedora, Ubuntu, Opensuse, Gentoo, Mandrake, Redhat, SUSE...)

WINDOWS
Follow the instructions on the Downloads page, under all-in-one installer.

MAC
Follow these instructions on the followwing page: https://wiki.gnome.org/action/show//Projects/GTK+/OSX/Building?action=show&redirect=GTK%252B%252FOSX%252FBuilding


6. 	Instructions to compile and run the project

No compilation needed.
Step one: Download Python 2.7 or higher 32 bit
Step two: Install PyGTK according to your operating system
Step three: Clone the repository (https://github.com/mcgill-ecse321/team-5.git) and do not change the folder organization
Step four: Double-click on "src/main.py" or for the command-line do the following:
(The following instructions assume that the user is currently in the folder "team-5")

cd src
python main.py
